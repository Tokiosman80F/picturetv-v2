# Website Refactoring Instructions

## Current Situation

This website was generated by Base44.
Currently everything exists in a single large HTML file.

## Goal

Refactor the website into a clean multi-page HTML/CSS/JS structure.

Do not convert to React or Next.js yet.

## Current Phase

Phase 1:

- Separate HTML pages
- Extract CSS
- Extract JavaScript
- Organize assets
- Maintain existing functionality

## Rules

- Do not delete existing features
- Preserve text content
- Preserve images and assets
- Keep code readable
- Use semantic HTML
- Avoid inline CSS
- Avoid inline JavaScript
- Create reusable components where possible
